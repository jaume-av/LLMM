# Resum del projecte API REST amb Express, SQLite, AlpineJS i SPA

## Objectiu del projecte

El projecte consisteix en crear una aplicació web tipus SPA (Single Page Application) utilitzant:

* Node.js
* Express
* SQLite
* Knex
* AlpineJS
* Fetch API
* Picnic CSS

L’aplicació permet gestionar:

* ciutats
* països
* províncies

mitjançant operacions CRUD.

---

# Estructura general del projecte

## Backend

El backend està desenvolupat amb Express.

Les rutes API es troben dins de:

```text
routes/
```

Per exemple:

```text
routes/ciutats.js
routes/paisos.js
routes/provincies.js
```

La connexió amb SQLite es realitza amb Knex des de:

```text
db.js
```

---

## Frontend

El frontend funciona com una SPA.

La vista principal està dins de:

```html
<div id="spa"></div>
```

Les diferents vistes es carreguen dinàmicament amb JavaScript.

Els fitxers principals són:

```text
public/js/spa.js
public/js/ciutats-vista.js
public/js/paisos-vista.js
public/js/provincies-vista.js
```

---

# Funcionament general de la SPA

La navegació es realitza canviant el contingut de:

```html
<div id="spa"></div>
```

Per exemple:

```javascript
function mostrarCiutats() {
    document.getElementById("spa").innerHTML = vistaCiutats();
    Alpine.initTree(document.getElementById("spa"));
}
```

Cada vegada que canviem una vista:

1. Inserim HTML nou dins de `spa`
2. Reinicialitzem AlpineJS

# GET

## Objectiu

Llegir dades de l’API i mostrar-les en una taula.

## Exemple de ruta

```text
GET /api/ciutats
```

## Exemple de backend

```javascript
router.get('/', function(req, res) {
    db('CIUTAT')
        .select('*')
        .then(function(rows) {
            res.json(rows);
        });
});
```

## Exemple de frontend

```javascript
fetch('/api/ciutats')
```

Les dades es guarden en:

```javascript
this.ciutats = dades;
```

# POST

## Objectiu

Crear nous registres.

## Ruta utilitzada

```text
POST /api/ciutats
```

## Backend

```javascript
router.post('/', function(req, res) {
    db('CIUTAT')
        .insert({
            NOM: req.body.NOM,
            DESCRIPCIO: req.body.DESCRIPCIO,
            IMATGE: req.body.IMATGE,
            POBLACIO: req.body.POBLACIO,
            PROVINCIA_ID: req.body.PROVINCIA_ID
        })
        .then(function(resultat) {
            res.json({ missatge: 'Ciutat creada correctament' });
        });
});
```

## Vista de creació

```javascript
function vistaNovaCiutat()
```

## App Alpine

```javascript
window.novaCiutatApp = function ()
```

## Enviament amb fetch

```javascript
fetch('/api/ciutats', {
    method: 'POST'
})
```

# PUT

## Objectiu

Modificar dades existents.

## Ruta utilitzada

```text
PUT /api/ciutats/:id
```

## Backend

```javascript
router.put('/:id', function(req, res) {
    db('CIUTAT')
        .where('ID', req.params.id)
        .update({
            NOM: req.body.NOM,
            DESCRIPCIO: req.body.DESCRIPCIO,
            IMATGE: req.body.IMATGE,
            POBLACIO: req.body.POBLACIO,
            PROVINCIA_ID: req.body.PROVINCIA_ID
        })
        .then(function(resultat) {
            res.json({ missatge: 'Ciutat modificada correctament' });
        });
});
```

## Vista de modificació

```javascript
function vistaModificarCiutat(id)
```

## App Alpine

```javascript
window.modificarCiutatApp = function (id)
```

## Carregar dades automàticament

```html
x-init="carregarCiutat()"
```

## Obtenir les dades

```javascript
fetch('/api/ciutats/' + id)
```

## Guardar els canvis

```javascript
fetch('/api/ciutats/' + id, {
    method: 'PUT'
})
```

## Botó Modificar

```html
@click="mostrarModificarCiutat(ciutat.ID)"
```

## Canvi de vista

```javascript
function mostrarModificarCiutat(id)
```

# DELETE

## Objectiu

Eliminar registres.

## Ruta utilitzada

```text
DELETE /api/ciutats/:id
```

## Backend

```javascript
router.delete('/:id', function(req, res) {
    db('CIUTAT')
        .where('ID', req.params.id)
        .delete()
        .then(function(resultat) {
            res.json({ missatge: 'Ciutat eliminada correctament' });
        });
});
```

## Botó Eliminar

```html
@click="eliminarCiutat(ciutat.ID)"
```

## Funció Alpine

```javascript
eliminarCiutat(id)
```

## Enviament DELETE

```javascript
fetch('/api/ciutats/' + id, {
    method: 'DELETE'
})
```

## Recarregar la taula

```javascript
this.carregarCiutats();
```

# AlpineJS

## x-data

Defineix quina app controla una secció HTML.

```html
<section x-data="ciutatsApp()">
```

---

## x-model

Relaciona un input amb una propietat JavaScript.

```html
<input x-model="ciutat.NOM">
```

---

## x-init

Executa una funció automàticament quan es crea la vista.

```html
x-init="carregarCiutat()"
```

---

## @click

Executa funcions d’AlpineJS.

```html
@click="eliminarCiutat(ciutat.ID)"
```
