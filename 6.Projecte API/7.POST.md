# POST

### Canvi de noms

Fins ara, alguns fitxers del backend i del frontend compartien el mateix nom.  

Per exemple:

```text
app.js                 (Express principal)
public/js/app.js       (frontend SPA)

routes/ciutats.js      (rutes backend)
public/js/ciutats.js   (frontend ciutats)

routes/paisos.js       (rutes backend)
public/js/paisos.js    (frontend països)

routes/provincies.js   (rutes backend)
public/js/provincies.js (frontend províncies)

```

> A çò funciona perque estan en carpetes diferents, però pot portar a confusió..


Per evitar-ho, canviarem el nom dels fitxers del frontend:


```text
public/js/app.js          -> public/js/spa.js

public/js/ciutats.js      -> public/js/ciutats-vista.js

public/js/paisos.js       -> public/js/paisos-vista.js

public/js/provincies.js   -> public/js/provincies-vista.js

```
