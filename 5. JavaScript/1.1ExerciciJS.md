# Exercicis JavaScript

## 1. Introducció a la consola

### 1. Obrir la consola

1. Obri qualsevol pàgina web (per exemple, google.com).
2. Prem **F12**.
3. Ves a la pestanya **Consola**.
4. Escriu:

```javascript
2 + 2
```

Prem **Enter**.

La consola mostrarà:

```
4
```

La consola és un lloc on podem escriure JavaScript i veure el resultat immediatament.

Tot el que escrius allí s’executa en el moment.

---

## Exercici guiat 1 – Expressions i variables

### Pas 1

Escriu en la consola:

```javascript
const preu = 50;
```

Prem Enter.

No apareix res perquè només hem declarat una variable.

Ara escriu:

```javascript
preu
```

Prem Enter.

La consola mostrarà:

```
50
```

La consola sempre mostra el valor de l’última expressió.

---

### Pas 2

Ara escriu:

```javascript
const descompte = 10;
```

Després:

```javascript
preu - descompte
```

La consola mostrarà:

```
40
```

Això és una expressió matemàtica.

---

### Pas 3

Ara guarda el resultat en una variable:

```javascript
const preuFinal = preu - descompte;
```

Comprova el valor:

```javascript
preuFinal
```

---

### Pas 4 – Condició

Ara escriu:

```javascript
preuFinal > 30
```

La consola mostrarà:

```
true
```

Això és una comparació.

Ara escriu:

```javascript
if (preuFinal > 30) {
    "Preu alt"
} else {
    "Preu baix"
}
```

La consola mostrarà:

```
"Preu alt"
```

En consola, si l’última línia és una cadena, la mostra automàticament.

---

## Exercici guiat 2 – Entrada i condició


### Pas 1 – Demanar una edat

Escriu:

```javascript
const edat = Number(prompt("Introdueix la teua edat:"));
```

S’obrirà una finestra.

Escriu un número i accepta.

Ara escriu:

```javascript
edat
```

Per comprovar el valor.

---

### Pas 2 – Fer una condició

Ara escriu:

```javascript
if (edat >= 18) {
    "Major d'edat"
} else {
    "Menor d'edat"
}
```

La consola mostrarà el resultat segons el número introduït.

---

### Pas 3 – Afegir una expressió

Escriu:

```javascript
edat + 5
```

I observa el resultat.

Estem combinant:

* Entrada
* Conversió
* Expressió matemàtica
* Condició


---


> **Important**
> Quan treballen en la consola del navegador:
>
> * Si volen **netejar la consola**, poden escriure:
>
> ```javascript
> clear()
> ```
>
> Açò **només elimina els missatges visibles de la consola**, però **les variables continuen existint**.
>
> * Si volen **reiniciar les variables**, han de **recarregar la pàgina**:
>
> ```
> F5
> ```

**Exercici 3 – Descompte en una botiga**

Fes el següent en la consola:

Demana a l’usuari la seua edat.
Demana si té targeta de client (pots usar `confirm()`).

Mostra:

“Té descompte” si té **65 o més** i **té targeta de client**.
“No té descompte” en qualsevol altre cas.

Condicions:

Has d’utilitzar `Number(prompt(...))`.
Has d’utilitzar `===`.
Has d’utilitzar `&&`.
