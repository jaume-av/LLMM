# Connexió a la base de dades i rutes GET

En esta segona part anem a fer el pas important del projecte:

```text
Express → SQLite → JSON real
```

Ara ja no retornarem dades escrites a mà, sinó dades reals de la base de dades.

---


Abans de connectar la base de dades, necessitem entendre una peça fonamental del projecte: **Knex**.

### Què és Knex

**Knex** és una llibreria de Node.js que ens permet treballar amb bases de dades utilitzant JavaScript.

És el que es coneix com un **query builder**.

Això vol dir que:

* no escrivim SQL directament (o no sempre)
* construïm les consultes amb JavaScript
* Knex genera el SQL per nosaltres

---

### Per a què serveix

Knex ens permet:

* connectar amb la base de dades
* fer consultes (`SELECT`)
* inserir dades (`INSERT`)
* actualitzar (`UPDATE`)
* eliminar (`DELETE`)

Per exemple, en SQL escriuríem:

```sql
SELECT * FROM PAIS;
```

Amb Knex escrivim:

```javascript
db('PAIS').select('*')
```

És més fàcil d’integrar dins del codi JavaScript, i ens proporciona una serie d'avantatges:

* evita escriure molt SQL
* és més llegible per a l’alumnat
* funciona amb diferents bases de dades (SQLite, MySQL, PostgreSQL…)
* s’integra perfectament amb Express

---

### Instal·lar Knex i SQLite

Ara necessitem instal·lar els mòduls necessaris.

Obri la terminal de WebStorm i executa:

```bash
npm install knex sqlite3
```

---

On `knex` és la llibreria que utilitzarem per fer consultes.

i `sqlite3` és el driver que permet a Node.js connectar amb bases de dades SQLite.

>Nota: Knex necessita este driver per funcionar amb SQLite.

---

Per conprovar que esta instal·lat:

Obri el fitxer:

```text
package.json
```

Hauries de veure alguna cosa semblant a:

```json
"dependencies": {
  "express": "...",
  "knex": "...",
  "sqlite3": "..."
}
```

## Creació el fitxer de connexió a la base de dades

En l’arrel del projecte (on està `app.js`), crea un nou fitxer:

```text
db.js
```

amb el següent contingut.

```javascript
const knex = require("knex");

const db = knex({
    client: "sqlite3",
    connection: {
        filename: "./ciutats.sqlite"
    },
    useNullAsDefault: true
});

module.exports = db;
```

Guarda el fitxer.

---

En el codi anterior del fitxer `db.js` l'usem per

**1.-  Carregar Knex**

```javascript
const knex = require("knex");
```

Carreguem la llibreria que ens permet fer consultes a la base de dades.

---

**2.- Configurar la connexió**

```javascript
const db = knex({
    client: "sqlite3",
    connection: {
        filename: "./ciutats.sqlite"
    },
    useNullAsDefault: true
});
```

Ací estem indicant:

* el motor de base de dades → `sqlite3`
* el fitxer → `ciutats.sqlite`
* configuració necessària per a SQLite

---

**3.- Exportar la connexió**

```javascript
module.exports = db;
```

Açò permet utilitzar `db` en altres fitxers, com per exemple en les rutes.

## Modificar el fitxer de rutes

Ara anem a treballar sobre:

```text
routes/index.js
```

> Este fitxer és on definirem totes les rutes GET

---

### Importar la base de dades

Al principi de `routes/index.js`, afegix:

```javascript
const db = require('../db');
```

---

### Ruta GET – tots els països

Afegix esta ruta dins de `routes/index.js`:

```javascript
router.get('/api/paisos', function(req, res) {
  db('PAIS')
    .select('*')
    .then(function(rows) {
      res.json(rows);
    })
    .catch(function(error) {
      res.status(500).json({ error: 'Error en la base de dades' });
    });
});
```

---

### Provar la ruta

```text
http://localhost:3000/api/paisos
```

* Exemple de resposta JSON

```json
[
  {
    "ID": 1,
    "NOM": "Espanya"
  },
  {
    "ID": 2,
    "NOM": "França"
  }
]
```

---

### Ruta GET per ID – països

Afegix:

```javascript
router.get('/api/paisos/:id', function(req, res) {
  db('PAIS')
    .where('ID', req.params.id)
    .first()
    .then(function(row) {
      res.json(row);
    });
});
```

---

* Al navegador posem:

```text
http://localhost:3000/api/paisos/1
```

i hauria d'eixir

```json
{
  "ID": 1,
  "NOM": "Espanya"
}
```

### Codi final de `routes/index.js`

```javascript
var express = require('express');
var router = express.Router();
const db = require('../db');

router.get('/', function(req, res) {
  res.send('Servidor funcionant');
});

router.get('/api', function(req, res) {
  res.send('OK');
});

router.get('/api/paisos', function(req, res) {
  db('PAIS').select('*').then(rows => res.json(rows));
});

router.get('/api/paisos/:id', function(req, res) {
  db('PAIS').where('ID', req.params.id).first().then(row => res.json(row));
});

router.get('/api/provincies', function(req, res) {
  db('PROVINCIA').select('*').then(rows => res.json(rows));
});

router.get('/api/provincies/:id', function(req, res) {
  db('PROVINCIA').where('ID', req.params.id).first().then(row => res.json(row));
});

router.get('/api/ciutats', function(req, res) {
  db('CIUTAT').select('*').then(rows => res.json(rows));
});

router.get('/api/ciutats/:id', function(req, res) {
  db('CIUTAT').where('ID', req.params.id).first().then(row => res.json(row));
});

module.exports = router;
```
