order

order reordena els elements VISUALMENT. El que té valor més baix apareix primer. Si dos elements tenen el mateix valor, es manté l'ordre original del HTML.
Orde natural (HTML)
A
B
C
D
Orde real del codi: A, B, C, D.
order aplicat (amb empats)
A
B
C
D
Valors aplicats:
• C → order: 1
• A → order: 2
• B → order: 2
• D → order: 5

Nou ordre visual segons el pes:
C, A, B, D

Nota important: A i B tenen el mateix ordre (2), així que es manté entre ells l'ordre HTML original: A va abans que B.
En resum:

• L’ordre natural és l’ordre del codi HTML.
• order només canvia la posició visual.
• Valor més baix = apareix primer.
• Valors iguals → es manté l’ordre HTML intern.
• Si no especifiques res: order = 0 per a tots.
