
# 14. Crear la connexió amb SQLite i Knex

Ara crea el fitxer `db.js` amb este contingut:

```javascript
const knex = require("knex");

const db = knex({
    client: "sqlite3",
    connection: {
        filename: "./ciutats.sqlite"
    },
    useNullAsDefault: true
});

module.exports = db;
```

---

# 15. Explicació del fitxer `db.js`

## Carregar Knex

```javascript
const knex = require("knex");
```

Açò importa la llibreria `knex`.

## Crear la connexió

```javascript
const db = knex({
    client: "sqlite3",
    connection: {
        filename: "./ciutats.sqlite"
    },
    useNullAsDefault: true
});
```

Ací indiquem:

* que el motor de base de dades és **SQLite**
* que el fitxer es diu `ciutats.sqlite`
* que volem utilitzar `useNullAsDefault: true`

## Exportar la connexió

```javascript
module.exports = db;
```

Açò permet utilitzar esta connexió des d’altres fitxers.

# 16. Crear la primera ruta real amb dades de la base de dades

Ara modifica `app.js` perquè utilitze la connexió a la base de dades.

## Nou `app.js`

```javascript
const express = require("express");
const db = require("./db");

const app = express();
const PORT = 3000;

app.get("/", function (req, res) {
    res.send("Servidor funcionant");
});

app.get("/api", function (req, res) {
    res.send("OK");
});

app.get("/api/paisos", function (req, res) {
    db("PAIS")
        .select("*")
        .then(function (rows) {
            res.json(rows);
        })
        .catch(function (error) {
            res.status(500).json({ error: "Error accedint a la base de dades" });
        });
});

app.listen(PORT, function () {
    console.log("Servidor escoltant en http://localhost:" + PORT);
});
```

---

# 17. Explicació de la ruta `/api/paisos`

## Importar la connexió

```javascript
const db = require("./db");
```

Açò carrega la connexió a SQLite.

## Definir la ruta

```javascript
app.get("/api/paisos", function (req, res) {
```

Esta ruta respondrà a les peticions GET fetes a:

```text
/api/paisos
```

## Fer la consulta

```javascript
db("PAIS").select("*")
```

Això consulta la taula `PAIS` i selecciona tots els camps de tots els registres.

## Retornar el resultat

```javascript
.then(function (rows) {
    res.json(rows);
})
```

Quan la consulta acaba correctament, el servidor retorna el resultat en format JSON.

## Gestionar errors

```javascript
.catch(function (error) {

```

Si ocorre algun error, el servidor respondrà amb codi `500` i un missatge d’error.

# 20. Errors habituals

## El servidor no arranca

Revisa que has escrit bé:

```javascript
const express = require("express");
```

i que has instal·lat Express amb:

```bash
npm install express
```

## La base de dades no es troba

Revisa que el fitxer:

```text
ciutats.sqlite
```

estiga a l’arrel del projecte.

## La ruta `/api/paisos` dona error

Revisa:

* que la base de dades existix
* que la taula es diu exactament `PAIS`
* que la connexió en `db.js` apunta bé al fitxer
